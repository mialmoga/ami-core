# AMI Core

AMI Core es el núcleo cognitivo local de un amigo digital persistente.

AMI piensa.
Otros sistemas actúan.

## Principios

- Local-first
- Offline por defecto
- Cuerpo intercambiable
- Persistencia > espectáculo

## Arquitectura (alto nivel)

AMI Core no depende de motores gráficos ni de servicios externos.
Se comunica con cuerpos (Unity u otros) mediante un contrato de intenciones y estados.

## Estado del proyecto

Fase: Preparación del núcleo  
AMI aún no actúa. Solo inicializa y existe.

## Auditoría

Todo cambio en este repositorio es auditado para mantener coherencia conceptual y técnica.
